# go-forum
## Руководство по администрированию приложения.
### Требования к ПО для возможности работы приложения:
* Любая ОС, для которой есть компилятор golang и клиент СУБД MySQL.
* Go 1.2 или выше.
* MySQL 4.1 и выше.
* Установлен GIT.
### Инструкция для развертывания приложения на localhost:
1. Необходимо установить драйвер для доступа к базе данных в $GOPATH, выполнив
команду go get github.com/go-sql-driver/mysql.
2. Создать в СУБД MySQL базу данных для форума.
3. Открыть файл All.sql в папке приложения. Заменить в первой строчке имя базы
данных на имя вашей базы данных в MySQL, снять комментарий. Заменить в
последней строке информацию об администраторе на свою, снять комментарий.
4. Выполнить в MySQL содержимое файла All.sql.
5. Открыть файл DataBase.go. Заменить в 10 строке значения переменных на
необходимые (имя пользователя в MySQL, пароль и название базы данных
соответственно).
6. Открыть файл Server.go. В предпоследней строке заменит первый аргумент
функции http.ListenAndServe на необходимый порт для развертывания приложения.
7. Выполнить в корневой папке приложения go build –o server Server.go
DataBase.go Tmplts.go.
8. Запустить файл server.

Чтобы остановить выполнение приложения необходимо остановить запущенный процесс.
Ни в коем случае нельзя удалять группы пользователей DEFAULT, DELETED, Admin, а
также пользователя DELETED. Данные действия повлекут неправильную работу всего
форума.
